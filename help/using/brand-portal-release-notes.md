# 릴리스 정보 {#release-notes}

Adobe Experience Manager Assets Brand Portal 2024.10.0 릴리스의 새로운 기능, 개선 사항, 해결된 중요한 문제 및 알려진 문제에 대한 인사이트를 얻으십시오.

## 릴리스 정보 {#release-information}

| 제품 | Adobe Experience Manager Assets Brand Portal |
|---|---|
| 버전 | 2024.10.0 |
| 날짜 | 2024년 10월 |

## 개요 {#overview}

Adobe Experience Manager(AEM) Assets Brand Portal을 사용하면 다양한 디바이스에서 승인된 크리에이티브 자산을 손쉽게 확보하고, 제어하고, 외부 당사자와 내부 비즈니스 사용자에게 안전하게 배포할 수 있습니다. 이렇게 하면 자산 공유의 효율성을 높이고, 자산의 시장 출시 기간을 앞당기고, 규정 비준수 및 무단 액세스의 위험을 줄이는 데 도움이 됩니다. Brand Portal을 이용하면 사용자는 언제 어디서나 회사에서 승인한 형식으로 자산을 탐색하고, 검색하고, 미리 보고, 다운로드하고, 내보낼 수 있습니다.

## 2024.10.0의 새로운 기능 {#whats-new-in-2024.10.0}

### 해결된 중요한 문제 {#critical-issues-fixed}

#### 버그 수정 {#bug-fixes}

이번 릴리스에는 다음과 같은 버그 수정 사항이 포함됩니다.

* 소싱 가져오기 이벤트에 대한 이메일 알림이 전송되지 않습니다.
* [!DNL Brand Portal]에서 검색 필터를 사용할 때 검색 필터 창 내에 중복된 폴더가 표시됩니다.
* 이름에 특수 문자가 포함되고 라이선스가 부여되지 않은 자산을 포함하는 컬렉션을 다운로드할 수 없습니다.
* [!UICONTROL 검색 양식 편집기]로 이동할 때 레이블이 현지화되지 않습니다.
* 링크 공유 창에서 레이블이 현지화되지 않습니다.
* 이름에 특수 문자가 포함된 비디오를 다운로드할 수 없습니다.
* [!DNL Adobe Experience Manager Assets]의 자산을 Brand Portal에 게시하거나 게시 취소하는 기능이 제대로 작동하지 않습니다.
* Brand Portal에서 다운로드한 비디오를 재생할 수 없습니다.

## 이전 릴리스

### 2024년 2월 릴리스 {#feb-2024}

**버그 수정 및 개선 사항**

이번 릴리스에는 다음과 같은 버그 수정 사항이 포함됩니다.

* 터키 로케일에서 DRM으로 보호되는 디지털 자산을 다운로드할 수 없습니다.
* 제목이 여러 줄인 자산이 포함된 기존 보고서를 열고 다운로드할 수 없습니다.
* 작업 표시줄에서 [!UICONTROL 다운로드] 버튼을 사용하여 자산을 다운로드할 때 최대 1000개의 자산만 다운로드됩니다.
* 콘텐츠 트리에서 볼 때 PSD 유형 자산의 이름이 잘못되었습니다.
* 자산 세부 정보 페이지의 [!UICONTROL 렌디션 삭제] 옵션이 작동하지 않습니다.
* 다운로드 팝업 창에서 자산의 제목과 크기가 잘못 정렬되었습니다.
* 보고서를 만들 때 레이블이 현지화되지 않습니다.
* Brand Portal에서 지원 관리자가 관리자라고 표시됩니다.

### 2023년 10월 릴리스 {#oct-2023}

**버그 수정 및 개선 사항**
이 릴리스에는 다음 개선 사항이 포함되어 있습니다.

* [!UICONTROL 컬렉션]을 탐색할 때의 성능이 향상되었습니다.

* OmniSearch 필드를 사용하여 부분 검색을 수행할 때의 검색 결과가 향상되었습니다.

이번 릴리스에는 다음과 같은 버그 수정 사항이 포함됩니다.

* [!UICONTROL 날짜] 및 [!UICONTROL 옵션] 설명을 [!UICONTROL 스마트 컬렉션]에 저장할 수 없습니다.

* 영어가 아닌 다른 로케일에서 작업하는 경우 [!UICONTROL 날짜 및 시간] 형식이 일관되지 않습니다.

* 자산 검색을 수행할 때 [!UICONTROL 삭제] 버튼이 표시되지 않습니다.

* [!UICONTROL 링크 공유]의 [!UICONTROL 제목] 필드에 멀티바이트 기호가 포함된 경우 보고서를 다운로드할 수 없습니다.

* PDF 유형 문서를 볼 때 레이블과 도구 설명이 현지화되지 않습니다.

### 2023년 8월 릴리스 {#aug-2023}

**버그 수정 및 개선 사항**
이 릴리스에는 다음 개선 사항이 포함되어 있습니다.

* [!UICONTROL 다운로드] 팝업에서 자산을 로드할 때의 성능이 향상되었습니다.
* 자산이나 자산의 렌디션을 다운로드하면 이제 zip 파일이 아닌 원래 파일 유형 형식으로 다운로드됩니다.

이번 릴리스에는 다음과 같은 버그 수정 사항이 포함됩니다.

* 레이블이나 태그가 길면 검색 필터에 적절하게 표시되지 않습니다.
* 다운로드 대화 상자에 긴 렌디션 이름을 표시할 수 없습니다.
* 카드 보기에서 비디오 자산을 미리 볼 수 없습니다.

### 2023년 5월 릴리스 {#may-2023}

**버그 수정**
이번 릴리스에는 다음과 같은 중요한 문제에 대한 수정 사항이 포함되어 있습니다.

* 공유 링크에서 자산을 다운로드하는 도중 오류가 발생하는 경우 오류 프롬프트의 `Notice` 및 `Close` 레이블이 현지화되지 않습니다.
* Brand Portal에서 `Filter` 창을 사용하여 검색 필터에 액세스하는 도중 **요청 헤더 필드가 너무 큼**&#x200B;이라는 오류가 표시됩니다.

**알려진 문제**
여기에는 다음의 알려진 문제가 포함됩니다.

* 자산 소싱 보고서 내용이 부분적으로 현지화되었습니다.
* 사용자 프로필의 일부 필드를 사용자 프로필에서 편집할 수 없습니다.

### 2023년 2월 릴리스 {#feb-2023}

**버그 수정**

이번 릴리스에는 다음과 같은 중요한 문제에 대한 수정 사항이 포함되어 있습니다.

* Brand Portal에서 프로필 사진을 업데이트할 수 없습니다.
* 콘텐츠 트리 창의 크기를 조절할 수 없습니다. 파일 이름이 콘텐츠 트리의 기본 폭보다 긴 경우 콘텐츠 트리를 가로 및 세로로 드래그할 수 없습니다. 결과적으로 긴 파일 이름을 읽을 수 없습니다.
* 검색 양식에서 동일한 속성 설명을 두 번 사용했음에도 불구하고 검색 결과가 일관되지 않습니다.
* 중간 로그인 페이지의 텍스트가 모든 언어로 현지화되어 있지 않습니다.

**개선 사항**

이번 릴리스에는 다음과 같은 개선 사항이 포함됩니다.

* 이제 PDF 자산의 미리보기가 개선된 새로운 최신 PDF 뷰어를 사용할 수 있습니다.
* 이제 관리자에게 보내는 자산 소싱 알림을 활성화하거나 비활성화할 수 있습니다. [!UICONTROL 일반 설정]으로 이동한 다음 [!UICONTROL `Notify Administrator of asset contribution`]을 활성화 또는 비활성화하십시오.

  ![관리자에게 자산 기여 알림](assets/notify-admin.png)

* 액세스 권한 요청이 비활성화된 경우, 승인되지 않은 사용자는 Brand Portal에 대한 액세스 권한을 요청할 수 없습니다.
* Brand Portal에 프로비저닝된 조직만 프로필 선택기 목록에 표시됩니다.

**알려진 문제**

여기에는 다음의 알려진 문제가 포함됩니다.

* 자산 소싱 보고서 내용이 부분적으로 현지화되었습니다.
* 사용자 프로필의 일부 필드를 사용자 프로필에서 편집할 수 없습니다.

### 2022년 10월 릴리스 {#oct-2022}

**해결된 중요한 문제**

이번 릴리스에는 다음과 같은 중요한 문제에 대한 수정 사항이 포함되어 있습니다.

* Brand Portal에서 서드파티 도구로 대용량 파일을 복사할 때 응답 시간이 느립니다.
* 렌디션 수 확인란을 선택하면 개별 렌디션을 선택하는 확인란이 비활성화됩니다.
* 검색 응답 시간이 느립니다.

>* 액세스 권한 요청 워크플로
>* 기여 폴더 공유
>* AEM으로 내보내기 시작
>* AEM으로 내보내기 완료

### 2022년 8월 릴리스 {#aug-2022}

**해결된 중요한 문제**

이번 릴리스에는 다음과 같은 중요한 문제에 대한 수정 사항이 포함되어 있습니다.

* NUI가 Experience Manager에서 자산을 처리하지 못하면 Brand Portal에 부정확한 자산 가져오기 상태가 표시됩니다.
* 미리 보기 작업이 실패하는 경우, 실패를 알리는 알림이 제공되지 않습니다.
* 각 자산의 `totalUploadedSize` 속성에 대한 부정확한 값이 수정되었습니다.
* 자산에 사용할 수 있는 렌디션이 많은 경우, **모든 항목 다운로드**&#x200B;를 클릭하면 Brand Portal이 잘못된 .ZIP 파일을 다운로드합니다.
* 일부 문자열의 번역이 Brand Portal 사용자 인터페이스에서 잘립니다.

### 2022년 5월 릴리스 {#may-2022}

**새로운 기능**

Brand Portal은 이제 12시간마다 자동 작업을 실행하여 AEM에 게시된 모든 Brand Portal 자산을 삭제합니다. 따라서 폴더 크기를 임계값 제한 이하로 유지하기 위해 기여도 폴더에 자산을 수동으로 삭제할 필요가 없습니다.

**해결된 중요한 문제**

이번 릴리스에는 다음과 같은 중요한 문제에 대한 수정 사항이 포함되어 있습니다.

* 색상 태그가 있는 자산이 포함된 폴더나 컬렉션을 다운로드하면 XML 파일도 함께 다운로드됩니다.
* 렌디션이 포함된 비디오를 다운로드하면 Brand Portal에서 잘못된 .ZIP 파일이 생성됩니다.
* AEM 작성자에서 사전 설정과 자산을 만든 다음 Brand Portal에 게시하면 자산을 다운로드하는 동안 동적 렌디션을 선택할 수 있습니다. 그러나 다운로드한 .ZIP 파일을 압축 해제할 수 없습니다. 이 문제로 인해 다운로드한 콘텐츠에 액세스할 수 없게 됩니다.
* Brand Portal에서 특정 폴더의 비디오 자산을 다운로드하는 도중 문제가 발생합니다.
* 이메일을 사용하여 기여 폴더의 URL을 공유하면 뷰어 및 편집자 역할이 경로를 사용하여 상위 폴더에 액세스하는 도중 문제가 발생합니다.
* 소싱 게시 보고서에 잘못된 작업 시작 시간이 표시됩니다.

### 2022년 2월 릴리스 {#feb-2022}

**새로운 기능**

* 게스트 사용자의 세션 시간 초과 임계값이 2시간에서 15분으로 단축되었습니다.
* 이제 사용자가 Adobe Document Cloud Viewer에서 PDF 페이지를 직접 볼 수 있으므로, 다중 페이지 PDF에 대해 추가로 제공되던 **[!UICONTROL 페이지 보기]** 옵션은 제거되었습니다.
* 사용자는 폴더를 검색하거나, 탐색하거나, 열 수 없습니다. 사용자 인터페이스에 `Failed to load data` 오류 메시지가 표시됩니다.
* **[!UICONTROL 렌디션]** 패널에는 Brand Portal에 게시된 자산의 모든 정적 렌디션이 나열되지 않습니다.
* **[!UICONTROL 렌디션]** 패널에는 자산의 스마트 자르기 렌디션이 나열되지만, 사용자는 스마트 자르기 렌디션을 미리 보거나 다운로드할 수 없습니다.
* 다운로드 대화 상자에는 선택한 자산의 스마트 자르기 렌디션이 나열되지만, 사용자는 스마트 자르기 렌디션을 다운로드할 수 없습니다.
* 관리자가 아닌 사용자는 자산을 다운로드할 때 원본 자산의 렌디션만 받게 됩니다. 시스템 및 사용자 정의 렌디션은 다운로드되지 않습니다.
* 검색 필터를 적용하여 자산을 다운로드할 때, 다운로드 대화 상자에서 `Download` 버튼이 비활성화되어 있으며 사용자는 자산을 다운로드할 수 없습니다.
* `Smart Tags` 또는 `Color Tags`가 활성화된 경우 다운로드 대화 상자에 `json` 파일이 렌디션으로 나열되고 이들 `json` 파일이 보관된 zip 폴더에 다운로드됩니다.
* 익명 사용자는 공유 링크를 사용하여 자산을 다운로드할 수 없습니다. 링크를 클릭하면 Brand Portal 로그인 페이지로 리디렉션되기 때문입니다.
* 시스템에 동시 활성 사용자 수에 대한 올바른 값이 반영되지 않습니다.
