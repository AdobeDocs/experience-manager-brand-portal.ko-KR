# 사용자 지정 검색 패싯 사용 {#use-custom-search-facets}

관리자는 검색 설명을 필터 패널에 추가하여 검색을 사용자 정의하고 검색 기능을 다용도로 만들 수 있습니다.

[!DNL Brand Portal] 필터 패널로 인해 승인된 브랜드 자산의 세부 검색을 [세밀하게](../using/brand-portal-searching.md#search-using-facets-in-filters-panel) [**검색할 수 있도록** 지원합니다](../using/brand-portal-searching.md#search-using-facets-in-filters-panel). 검색 패싯은 관리 도구에서 **검색 양식을** 통해 필터 패널에 사용할 수 있습니다. 에셋 관리자 검색 레일이라는 기본 검색 양식이 관리 도구의 검색 양식 페이지에 있습니다. 하지만 관리자는 검색 설명을 추가, 수정 또는 제거하여 기본 검색 양식 (자산 관리자 검색 레일) 를 편집하여 기본 필터 패널을 사용자 지정할 수 있으므로 검색 기능을 다용도로 만들 수 있습니다.

다양한 검색 설명을 사용하여 **필터** 패널을 사용자 지정할 수 있습니다. 예를 들어, 이 설명서에서 지정하는 단일 속성과 일치하는 자산을 검색하려면 속성 설명을 추가합니다. 특정 속성에 대해 지정하는 하나 이상의 값과 일치하는 자산을 검색하려면 옵션 설명을 추가합니다. 지정된 날짜 범위 내에서 만든 자산을 검색하려면 날짜 범위 설명을 추가합니다.

>[!NOTE]
>
>[!DNL AEM] 조직에서 [[! DNL AEM] 로](../using/publish-schema-search-facets-presets.md#publish-search-facets-to-brand-portal) [!DNL Brand Portal][!DNL Brand Portal]변경했습니다.

## 검색 설명 추가 {#add-a-search-predicate}

**필터** 패널에 검색 설명을 추가하려면:

1. 관리 도구에 액세스하려면 맨 위의 도구 모음에서 [!DNL AEM] 로고를 클릭합니다.

   ![](assets/aemlogo.png)

2. [관리 도구] 패널에서 [양식 **검색] 를 클릭합니다**.

   ![](assets/navigation-panel-1.png)

3. 양식 **검색** 페이지에서 **자산 관리 검색 레일을 선택합니다**.

   ![](assets/search-forms-page.png)

4. 상단에 표시되는 도구 모음에서 **편집을** 클릭하여 검색 편집 양식을 엽니다.

   ![](assets/edit-search-form-1.png)

5. 검색 양식 **편집** 페이지의 설명 **선택** 탭에서 기본 창으로 설명을 드래그합니다. 예를 들어 **속성 설명을 드래그합니다**.

   **속성** 필드는 기본 창에 나타나며 오른쪽의 **설정** 탭에는 속성 예측이 표시됩니다.

   ![](assets/partial-prop-predicate.png)

   >[!NOTE]
   >
   >**설정** 탭의 머리글 레이블은 선택하는 설명 유형을 식별합니다.

6. **[설정** ] 탭에서 속성 설명 레이블, 자리 표시자 텍스트 및 설명을 입력합니다.

   * 지정된 속성 값을 기반으로 자산의 구문 검색 (및 와일드카드 검색) 를 허용하려면 **부분 검색을**&#x200B;선택합니다. 기본적으로, 설명자는 전체 텍스트 검색을 지원합니다.
   * 속성 값을 기반으로 자산 검색을 대/소문자를 구분하지 않으려면 대/소문자 ****&#x200B;무시를 선택합니다. 기본적으로 검색 필터에서 속성 값 검색은 대/소문자를 구분합니다.
   >[!NOTE]
   >
   >**부분 검색** 확인란을 선택할 때 대/소문자를 **무시하면 기본적으로 선택됩니다** .

7. **속성 이름** 필드에서 속성 선택기를 열고 수행되는 검색을 기반으로 속성을 선택합니다. 또는 속성의 이름을 입력합니다. `  jcr :content/metadata/dc:title``./jcr:content/metadata/dc:title`예를 들어 또는을 입력합니다.

   ![](assets/title-prop.png)

8. **완료를** 클릭하여 설정을 저장합니다.
9. **자산** 사용자 인터페이스에서 오버레이 아이콘을 클릭하고 **필터를** 선택하여 **필터** 패널로 이동합니다. **속성** 설명이 패널에 추가됩니다.

   ![](assets/property-filter-panel.png)

10. **속성** 텍스트 상자에서 검색할 자산의 제목을 입력합니다. 예를 들어 "Adobe" 를 입력합니다. 검색을 수행하면 검색 결과에 제목이 "adobe" 인 자산이 표시됩니다.

## 검색 설명 삭제 {#delete-a-search-predicate}

검색 설명을 삭제하려면 다음 단계를 수행합니다.

1. Adobe 로고를 클릭하여 관리 도구에 액세스합니다.

   ![](assets/aemlogo.png)

2. [관리 도구] 패널에서 [양식 **검색] 를 클릭합니다**.

   ![](assets/navigation-panel-2.png)

3. 양식 **검색** 페이지에서 **자산 관리 검색 레일을 선택합니다**.

   ![](assets/search-forms-page.png)

4. 상단에 표시되는 도구 모음에서 **편집을** 클릭하여 검색 편집 양식을 엽니다.

   ![](assets/edit-search-form-2.png)

5. 검색 양식 **편집** 페이지의 주 창에서 삭제할 설명을 선택합니다. 예를 들어 **속성 설명을 선택합니다**.

   오른쪽의 **설정** 탭에는 속성 설명 필드가 표시됩니다.

6. 속성 설명을 삭제하려면 저장소 아이콘을 클릭합니다. 필드 **삭제** 대화 상자에서 **삭제를** 클릭하여 삭제 작업을 확인합니다.

   속성 **설명** 필드가 주 창에서 제거되고 **설정** 탭이 비어 있게 됩니다.

   ![](assets/search-form-delete-predicate.png)

7. 변경 내용을 저장하려면 도구 모음에서 **완료를** 클릭합니다.
8. **자산** 사용자 인터페이스에서 오버레이 아이콘을 클릭하고 **필터를** 선택하여 **필터** 패널로 이동합니다. **속성** 설명이 패널에서 제거됩니다.

   ![](assets/property-predicate-removed.png)
